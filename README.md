# RPS---Ultimate-Battle-

1. Run this command in your terminal to install necessary packages.
    - pip3 install -r requirements.txt

2. Run the game qith this command (For Now only single player is available).
    - python game_single.py

3. Draw your character! Make a shape that looks like rock, paper, or scissors and add anything you want to it! Then click enter to battle.
    - you can use different colors by clicking the letters B (Blue), R (Red), Y (Yellow), W (White), K (Black) etc.
    - If you want to exit click esc

4. Fight to the death! (For now the defaul enemy is scissors run by AI)
    - Move with WASD and attack with space (Right not attacks may be temporarily buggy)
